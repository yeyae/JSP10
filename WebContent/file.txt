파일 업로드
대부분의 경우에 파일은 데이터베이스에 직접 저장하지 않습니다.
서버 컴퓨터의 공간에 따로 저장을하고, 데이터베이스에는 저장된 파일의 정보만 가지고 있음.

파일을보면 크기 byte
데이터베이스에 넣을때 결국이걸 통째로 넣긴 힘들고 byte단위 쪼개서 넣게 됩니다.

1. 파일의 정보를 저장할 테이블 생성
2. 파일 업로드 요청을 받으면 실제 파일을 복사해서 저장하고,
3. 복사한 파일의 정보를 테이블에 저장

저장할 파일의 정보 ==> 파일 이름, 파일을 저장한 시간

create table uploadfile(
	filename varchar2(100) primary key,
	writedate date default sysdate
)
